# 🟠 PROJECT_BLUEPRINT.md - Master Plan

## Project Overview
A high-end, AI-powered portfolio website for **Tarun Mankar**, a Gen AI Content Developer. The site showcases mobile apps, AI-generated videos, music, and software development expertise.

## Domain Info
- **Primary Domain**: `https://tarun-mankar.web.app/`
- **Hosting**: Firebase Hosting

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS3, Vanilla JavaScript
- **Design System**: Glassmorphism, Dark Mode, Fluid Animations
- **Assets**: Optimized WebP images, Font Awesome 6.4.0, Google Fonts (Outfit)
- **SEO**: Sitemap.xml, Robots.txt, JSON-LD Schema, Open Graph tags

## Directory Structure
- `/`: Main Portfolio (Home, About, Expertise, Projects, Socials)
- `/splitbhai/`: Landing page & Privacy Policy for SplitBhai App
- `/smartemi/`: Landing page & Privacy Policy for SmartEMI App
- `style.css`: Global styles and design tokens
- `script.js`: Global logic and animations

## Design Tokens
- **Background**: `#0a0a0f`
- **Primary**: `#6366f1` (Indigo)
- **Secondary**: `#ec4899` (Pink)
- **Accent**: `#06b6d4` (Cyan)
- **Glass**: `rgba(255, 255, 255, 0.03)` with `12px` blur
