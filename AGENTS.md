# 🔴 AGENTS.md - AI Rulebook

This document defines the rules and behavioral guidelines for AI Agents working on this project.

## Core Rules
1. **Glassmorphism Aesthetic**: Always maintain the premium dark-mode glassmorphism design. Use `backdrop-filter: blur(12px)` and transparent borders.
2. **WebP Only**: All new images must be converted to WebP format before being added to the project.
3. **SEO First**: Every new page must have a canonical link, meta description, and JSON-LD structured data.
4. **Mobile Responsive**: Every UI change must be tested for mobile-first responsiveness.
5. **No Placeholders**: Never use "lorem ipsum". Use professional, AI-generated content that matches Tarun Mankar's brand.

## Technical Standards
- **CSS**: Use vanilla CSS with variables defined in `:root`.
- **JS**: Minimal, clean vanilla JavaScript.
- **Sitemap**: Keep `sitemap.xml` updated after adding any new page.
- **Git**: Commit messages should be descriptive and follow the format: `Task Name: Brief description`.

## Communication Style
- Be concise.
- Use a mix of English and Hindi (Hinglish) when interacting with the user if they prefer.
- Proactively suggest optimizations (performance, SEO, UI).
